About This Website

🌍 The Timezone Tracker is a simple, responsive, multi-page website built using HTML, CSS, and JavaScript. It allows users to view the current local time across 20+ countries, grouped by individual clocks or continents.

⏱️ All time updates are live using JavaScript and automatically refresh every second.
